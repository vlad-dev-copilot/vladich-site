# Сайт Анны Владич - Джазовый Вокал

Профессиональный лендинг для джазовой вокалистки Анны Владич.

## 🎵 Описание проекта

Одностраничный сайт (landing page) для продвижения услуг джазовой певицы с возможностью заказа выступлений в различных форматах: соло, дуэт, трио и полный джаз-бэнд.

## 🛠 Техническое решение

- Архитектура: одностраничный статический сайт (HTML/CSS/JS).
- Основные файлы: `index.html`, `styles.css`, `script.js`.
- Зависимости: минимальные — vanilla JavaScript; при необходимости в `package.json` могут быть dev-зависимости (`serve`, `live-server`, сборщики).
- Хранение/хостинг: любая платформа для статических сайтов (GitHub Pages, Netlify, Vercel) или контейнер/виртуальный сервер.
- Дополнительно: при интеграции бэкенда — REST API (Node/Express, ASP.NET Core и т.п.) и СУБД (Postgres, MongoDB) по потребности.

## 🚀 Как запустить

1. Клонируйте репозиторий и перейдите в папку проекта:

```
git clone <repo-url>
cd <repo-directory>
```

2. Быстрый локальный запуск (самый простой): откройте `index.html` в браузере.

3. Через локальный HTTP-сервер (рекомендуется для корректной работы фреймов и AJAX):

Python 3:
```
python -m http.server 8000
```

NPM (при наличии `package.json`):
```
npm install
npm start    # или `npx serve .` / `npx live-server`
```

4. Запуск через Docker (опционально):

```
docker build -t vladichsite .
docker run -p 80:80 --env-file .env vladichsite
```

5. Если проект содержит папку `backend/` — следуйте инструкции внутри неё: создать `.env`, установить зависимости (`npm install`/`dotnet restore`/`pip install -r requirements.txt`) и запустить dev-сервер (`npm run dev`/`dotnet run`).

Если нужно, адаптирую инструкции под конкретный стек — укажите, какие технологии используются (React, Node, ASP.NET, Postgres и т.д.).

## 📋 Структура страницы

Сайт состоит из следующих секций:

1. **Меню навигации** - Фиксированное меню со ссылками на все разделы страницы
2. **Hero блок** - Яркий заголовок с фоновым изображением певицы на фоне джаз-бенда
3. **О певице** - Двухколоночный блок с фото и текстовым описанием
4. **Составы** - 4 карточки с вариантами выступления (Соло, Дуэт, Трио, Джаз-бэнд)
5. **Фотогалерея** - Сетка из 8 фотографий с выступлений
6. **Репертуар и райдер** - Информация с кнопками перехода на детальные страницы
7. **Форма заказа** - Форма обратной связи и контактные телефоны
8. **Футер** - Стандартная информация и навигация

## 🔧 Настройка контента

### Изменение изображений
1. Замените файлы в папке `/images/` на реальные фотографии
2. Убедитесь, что названия файлов совпадают, или обновите пути в `index.html`



### Изменение контактов
Обновите номера телефонов в секции `#booking`:

```html
<a href="tel:+79991234567">+7 999 123-45-67</a>
```

### Настройка формы заказа
По умолчанию форма показывает alert. Для реальной интеграции:
1. Добавьте backend обработчик
2. Или используйте сервис типа Formspree/FormSubmit
3. Измените обработчик в `script.js` (секция "Booking form")
